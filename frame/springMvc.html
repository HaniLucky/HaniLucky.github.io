* 重定向跳转到公网地址

@RequestMapping
public class IndexController {
    @GetMapping({"/", "/index"})
    public ModelAndView index() {
       return new ModelAndView(new RedirectView("https://www.baidu.com"));
    }
}

请求转发和重定向的区别

# 请求转发和重定向的区别

# 请求转发

  • 请求次数

1次

  • url是否变更

url不变化,请求路径必须是服务内部的一个地址

  • 适用场景

应用内部的转发

# 重定向

  • 请求次数

2次

  • url是否变更

url变化,变化为重定向的地址,可以重定向到外网

  • 适用场景

重定向到外网

# 返回静态页面

html 页面位置:resources /templates

可能是springboot 高版本的需要这样配置,以前只需要编写一个页面跳转接口就可以了

# 编写通用的页面跳转接口

@Controller
@RequestMapping("/page")
public class PageController {
    @GetMapping("/{pageName}")
    public String page(@PathVariable("pageName") String pageName){
        return pageName;
    }
}
